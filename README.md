# The Sports Hub

A premium, pixel-perfect sports dashboard built with React, TypeScript, and Tailwind CSS.

## üöÄ Features

- **Fixtures Dashboard**: Real-time match listings with live status indicators.
- **Match Details**: In-depth view of match events with a vertical timeline.
- **Polling System**: Automatic background updates for live scores every 20 seconds.
- **Premium UI**: Dark mode design with glassmorphism, custom scrollbars, and smooth animations.
- **Responsive Layout**: Built using Tailwind CSS for a seamless experience.

## üì¶ Installation

1.  **Clone the repository**:

    ```bash
    git clone <repository-url>
    cd the-sports-hub
    ```

2.  **Install dependencies**:

    ```bash
    npm install
    ```

3.  **Run the development server**:

    ```bash
    npm run dev
    ```

4.  **Open in browser**:
    Navigate to `http://localhost:5173`

## üìñ API Usage

This project connects to the public [The SportsDB API](https://www.thesportsdb.com/).

- Next Events: `/eventsnext.php?id=133602`
- Event Lookup: `/lookupevent.php?id={ID}`
